# DarkcraneStore 项目需求规格说明书 (PRD)

## 1. 项目概述 (Project Overview)
本项目旨在打造一个创新的 VPS 拼车/合租服务平台，主要服务于中国大陆有特定网络加速需求的用户。
平台引入了新颖的**“汽车与加油”**创意概念，将枯燥的 VPS 服务器参数包装为不同级别的“车辆”和“燃油”，极大地降低了用户的理解门槛并增加了趣味性。

- **核心模式**：支持 2-5 人灵活拼车，满员自动发车，也支持单人全包购买多个车位（Seat）。
- **交付与售后**：支持交付 VPS 原始管理权限；针对跳车情况（中途退租）不支持退款，保证车上其他成员权益不受损。
- **续费策略**：支持常规月付。部分特价月付车型定义为“月抛机”，不支持续费。
- **动态补员**：发车后若有人中途下车，平台支持即时“中途补人”，费用按剩余时间精准结算。

## 2. 核心系统与页面 (Core Systems & Pages)

### 2.1 门户首页 (Portal Homepage)
- **视觉动态**：采用动态背景设计（如后排第一视角的行驶动画），营造沉浸式体验。
- **核心交互**：提供醒目的“快速上车”按钮，一键直达选购页面。
- **数据看板 (预览面板)**：实时展示平台的核心运营数据（总车辆数、已发车数、空车待客数）。

### 2.2 上车选购页 (Vehicle Selection Page)
展示平台提供的全线 VPS 实例，按线路质量和性能划分为 5 个级别的“车型”：
1. **A 级车**：普通线路（BGP，无专门优化）。晚高峰较拥挤，主打极高性价比。
2. **B 级车**：单程优化线路（去程/回程之一普通优化）。晚高峰波动一般，价格适中。
3. **C 级车**：双程优化线路（双向普通优化）。晚高峰波动较小，定价偏高。
4. **D 级车**：精品专线（双程精品优化 + SLA 保护 + 抗 DDOS）。全天候基本无波动，附赠原生 ISP 落地出口，价格极高。
5. **SuperCar & HyperCar**：终极方案（高性能独立硬件 + 精品线路 + 无限流量家宽出口 + 专享独占 IP）。仅支持高级定制和页面展示。

### 2.3 真家宽专区 (True Residential Broadband Zone)
为高级 AI 用户和流媒体重度用户打造的高端专区（定价 80-150元/Seat/月）。
- **覆盖地区**：港、台、英、美及部分亚太核心地区。
- **核心卖点**：完美解锁全球所有流媒体平台限制；确保 Claude、Codex、Gemini 等敏感 AI 服务的对话不降智、账号防封禁。

## 3. 创意概念体系 (Creative Concepts)

### 3.1 车辆命名映射 (Naming Conventions)
利用现实中的汽车品牌与车型，巧妙映射 VPS 的物理所在地：
- **香港地区 VPS**：映射为“中国特供合资车”或“国产旗舰”（例如 C级车命名为 A6L、530Li、E300L、红旗H9 等）。
- **欧美地区 VPS**：映射为“原装进口车”（例如 Giulia、430、XT5 等）。
*(核心规则：车型原产国/品牌基调与 VPS 物理地址严格对应)*

### 3.2 充值与消耗系统 (Fuel System)
平台摒弃传统的“套餐费”概念，采用“加油”作为充值和计费的包装：
- **92号油**：适用于 A、B 级车，单价 5-10 元 / Seat / 月。
- **95号油**：适用于 C 级车，单价 12-20 元 / Seat / 月。
- **98号油**：适用于 D 级车，单价 25-40 元 / Seat / 月。
- **特殊燃油**：SuperCar & HyperCar 使用专属定制燃油。

## 4. UI/UX 视觉规范 (Art Direction)
- **主题色调**：全局采用极致的暗色调设计（Dark Mode Only），不提供明亮模式。
- **美术风格**：赛博朋克 (Cyberpunk) 风格，注重霓虹光感、毛玻璃特效（Glassmorphism）和科技感 UI 组件。

## 5. 账户与支付模块 (Account & Payment)

### 5.1 注册与验证
- 强制要求**邮箱验证**。
- 坚决**不需要**实名认证，保护用户隐私。

### 5.2 拼车与支付逻辑
- **发车条件**：默认逻辑为“人满发车”，满足规定人数后方可发起最终扣款。
- **买断机制**：若单人购买多个 Seat（包车），需预付多出的空置 Seat 费用以强制发车。
- **支付网关支持**：
  1. **USDT 加密货币支付**（主推及首选推荐方式）。
  2. 支付宝 (Alipay)。
  3. 银行卡支付（限定多币种卡，且由于通道成本，需额外征收 2% 手续费）。
